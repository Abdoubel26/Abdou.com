## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Abdoubel26/Abdou.com.git
cd Abdou.com

# Install dependencies
npm install

# Start the development server
npm run dev
```

The portfolio will be available at `http://localhost:3000`
