# 🎨 Abdou Belounis — Developer Portfolio

Welcome to my personal developer portfolio! A clean, minimal, and interactive showcase of my work, expertise, and the books that have shaped my perspective.

**🌐 Live Site**: [abdoubelounis.netlify.app](https://abdoubelounis.netlify.app)

---

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Abdoubel26/Abdou.com.git
cd Abdou.com

# Install dependencies
npm install

# Start the development server
npm run dev
```

The portfolio will be available at `http://localhost:5173`
